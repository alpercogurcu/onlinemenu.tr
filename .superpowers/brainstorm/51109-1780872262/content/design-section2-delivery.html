Bölüm 2 — Order Channel & Delivery Flow

Sipariş kanalı modeli · Inbound delivery akışı · Outbound catalog sync

Orders Tablosu — Eklenen Alanlar
order_channel TEXT NOT NULL -- 'dine_in' | 'takeaway' | 'click_collect' | 'delivery'
delivery_integrator_id UUID | NULL -- FK → branch_delivery_integrators
external_order_id TEXT | NULL -- Yemeksepeti/Getir sipariş no
external_order_ref JSONB | NULL -- platform'dan gelen ham veri (audit)
delivery_address JSONB | NULL -- kurye adresi
channel_commission NUMERIC -- platform komisyonu (raporlama)
accept_deadline_at TIMESTAMPTZ | NULL -- zaman limitli kabul için
accepted_at TIMESTAMPTZ | NULL -- kim/ne zaman kabul etti
accepted_by UUID | NULL -- user_id | NULL = oto-kabul
accepted_by = NULL → oto-kabul (timer doldu). accepted_by = user_id → kasiyerin manuel kabulü. Her ikisi de raporlanabilir.
Inbound — Yemeksepeti'nden Sipariş Gelişi
YS
Webhook Alındı
POST /webhooks/delivery/{integrator_id}
platform/integration → imza doğrulama (HMAC) → delivery.order.received.v1 NATS event
N
Delivery Consumer — POS Order Oluşturulur
delivery modülü event'i tüketir → orders tablosuna yazar
order_channel = 'delivery', delivery_integrator_id dolu, accept_deadline_at = now() + 60s
status = 'pending_accept'
📱
WebSocket Bildirimi — Sadece "kasa" Rolündeki Cihazlar
ws://api-pos/branch/{id}/role/kasa → bildirim gönderilir
Garson tabletleri, KDS ekranı bu bildirimi almaz.

KDS
Mutfak Ekranına Düşer — Kanal Bilgisi ile
status = 'preparing'
KDS ekranında kart başlığı: 🛵 Yemeksepeti  |  🚀 Getir  |  🟠 Trendyol
Mutfak fişinde platform adı basılır → kurye pakete zımbalayabilir.
✓
Teslim & Kapatma
Kurye geldi → status = 'delivered' → pos.order.completed.v1 event
Billing consumer faturayı tetikler. Ciro raporuna channel = delivery / provider = yemeksepeti olarak girer.
Outbound — Delivery Platformlarına Catalog Sync
catalog.product.updated.v1
Ürün fiyatı/adı değişti
→ YS + Getir + Trendyol (broadcast)
catalog.product_availability.changed.v1
Ürün satışa kapatıldı/açıldı
→ sadece delivery kanalları
tenant.branch_hours.updated.v1
Çalışma saatleri değişti
→ YS + Getir + Trendyol
catalog.image.updated.v1
Ürün görseli güncellendi
→ MinIO URL → platformlara push
⚠️ Platform sınırlaması: Her delivery platformunun kendi rate limit'i var (Yemeksepeti: ~10 req/sn). platform/integration.RateLimiter bunu yönetir. Aynı anda 50 ürün güncellendiyse → batch queue → sırayla gönderilir.
